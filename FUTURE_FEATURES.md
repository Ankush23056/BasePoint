# Future Features & Product Roadmap

BasePoint is designed to be a privacy-first, high-performance financial coach. Below are the planned upgrades to transition from a local-first dashboard to a comprehensive wealth management suite.

## 🚀 Upcoming Enhancements

### 1. Cloud Sync & Multi-Device Support
- **End-to-End Encrypted Sync**: Using Supabase or Firebase to allow users to access their data across multiple devices without compromising privacy.
- **Biometric Lock**: Adding FaceID/TouchID support for mobile PWA users.

### 2. Intelligent Automation
- **AI Expense Categorization**: Using LLMs to automatically categorize transactions based on merchant names.
- **Predictive Burn Rate**: Forecasting when a user will run out of "Daily Spending Power" based on historical habits.

### 3. Multi-Currency & Global Support
- **Real-time FX Rates**: Automatically converting foreign expenses into the user's primary currency (e.g., USD to INR).
- **Localized Formatting**: Support for various currency symbols and numbering systems (Millions vs. Lakhs).

### 4. Collaborative Finance
- **Shared Wallets**: Allowing partners or roommates to manage a shared budget while keeping personal transactions private.
- **Budget Alerts**: Push notifications when a category exceeds 90% of its guardrail.

### 5. Advanced Reporting
- **PDF/CSV Export**: Monthly financial health reports for tax or accounting purposes.
- **Year-over-Year (YoY) Analysis**: Visualizing long-term wealth growth trends.

---
*Developer Note: These features are prioritized based on user feedback and the goal of maintaining a lightweight, performant "Base" for all financial operations.*
